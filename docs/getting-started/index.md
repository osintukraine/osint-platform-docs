# Getting Started

Welcome to the OSINT Intelligence Platform documentation. This guide will help you understand the platform, get it running, and start monitoring Telegram channels for OSINT intelligence.

## What You'll Learn

This Getting Started section covers:

- [**What is the OSINT Platform?**](what-is-osint-platform.md) - Understand what the platform does, who it's for, and what makes it different from other OSINT tools.

- [**Quick Start Guide**](quick-start.md) - Get the platform running in under 10 minutes with step-by-step instructions.

- [**Core Concepts**](concepts.md) - Learn the essential terminology and concepts that power the platform.

## Who This Is For

### New Users
Start with [What is the OSINT Platform?](what-is-osint-platform.md) to understand the system's capabilities, then follow the [Quick Start Guide](quick-start.md) to get it running.

### Experienced OSINT Analysts
Jump straight to the [Quick Start Guide](quick-start.md) if you already understand OSINT workflows and want to deploy the platform immediately.

### Operators & Administrators
After completing the Quick Start, explore the [Operator Guide](../operator-guide/index.md) for detailed operational workflows including channel management, monitoring, and backup procedures.

## Prerequisites

Before you begin, make sure you have:

- **Docker & Docker Compose** installed on your system
- **8GB+ RAM** (16GB recommended for AI models)
- **Telegram API credentials** from [my.telegram.org](https://my.telegram.org/apps)
- **Basic command line familiarity**
